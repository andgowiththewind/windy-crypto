# 介绍

`windy-crypto`是一个基于`SpringBoot+Vue`的可视化文件加解密工具，支持对文件进行加密、解密等操作。加密方式采用流式加密，每个字节都加密，破解难度=256^字节数，基本上8个字节以上的文件在当前世界的物理条件已经无法破解。
